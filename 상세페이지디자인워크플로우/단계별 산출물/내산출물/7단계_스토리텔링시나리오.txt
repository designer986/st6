## 2️⃣ 스토리 전개 흐름 요약

> **후킹(고객 상황 공감)** → **공감(페인포인트 확인)** → **해결(착용감·디자인·범용성)** → **증명(소재·리뷰·A/S)** → **감성 연결(브랜드 가치)** → **비교(사이즈 명확화)** → **CTA(구매 유도)**
>
> **감정 여정 설계:**  
> 1. "매일 아침 고민" (공감)  
> 2. "무거운 귀걸이 문제" (불편 확인)  
> 3. "편안함+디자인+범용성 3가지 해결" (솔루션)  
> 4. "실버 925 + 고객 후기" (신뢰 구축)  
> 5. "3040 커리어우먼의 선택" (정체성 부여)  
> 6. "지금 구매" (행동 유도)
>
> **경쟁사 대비 차별점:**  
> - 경쟁사: 제품 디테일 중심의 고급스러운 이미지 나열  
> - AURA SILVERE: **고객의 일상 문제 → 기능적 해결 → 감성적 가치** 순서로 설득력 강화

## 5️⃣ 섹션별 상세 카피 (이미지 구성 가이드 포함)

### 📌 섹션 1: 메인 후킹 + 페르소나 공감
**[이미지: 모델 착용 풀샷 - 오피스 캐주얼 룩]**

```
매일 아침, 귀걸이 고민 끝.
365일 정착템이 될 단 하나의 링.

출근부터 저녁 모임까지,
바쁜 아침 귀걸이 바꿀 시간 없는 당신을 위한
AURA SILVERE 실버 블랑 링.
```

**설득 요소:** 후킹 + 공감  
**데이터 반영:** 3단계 리뷰("365일 착용", "데일리 정착템") + 1단계 페르소나(바쁜 커리어우먼)

---

### 📌 섹션 2: 문제 제기 (공감 강화)
**[이미지: 귀를 만지는 모델 - 불편함 표현]**

```
예쁜 귀걸이는 왜 무겁죠?
장시간 착용하면 귀가 아픈 건 당연한 걸까요?

아니요, 당연하지 않습니다.
```

**설득 요소:** 공감  
**데이터 반영:** 3단계 부정 키워드("무거워서 귀 아파요")

---

### 📌 섹션 3: 핵심 해결 제시 (3가지 USP)
**[이미지 1: 귀에 착용된 클로즈업 - 편안한 핏감]**

```
1. 하루 종일 편안함

원터치 방식의 간편함,
최적 무게 밸런스로 귀 부담 제로.
안 빼고 자도 변형 없는 안정감.
```

**[이미지 2: 웨이브 디테일 클로즈업]**

```
2. 절제된 볼드함

밋밋하지 않지만 튀지 않는,
유려한 웨이브 곡선이 만드는 모던한 존재감.
단정함 속의 세련된 포인트.
```

**[이미지 3: 다양한 액세서리와 레이어드 착용샷]**

```
3. 실버·골드 고민 끝

투톤 믹스로 모든 액세서리와 자연스러운 매치.
목걸이, 팔찌 색상 상관없이
TPO 초월한 만능 키.
```

**설득 요소:** 해결  
**데이터 반영:** 6단계 USP 3가지 + 2단계 FACT/EMOTION + 3단계 긍정 키워드

---

### 📌 섹션 4: 증명 (소재 + 고객 후기)
**[이미지: 실버 925 각인 클로즈업]**

```
실버 925, 믿을 수 있는 선택

변색 걱정 없는 무도금,
알레르기 안전 소재로 오래도록 착용 가능.

"휘뚜루마뚜루 착용 중이에요"
"안 빢고 자도 변형 없고, 어떤 옷이든 다 잘 어울려요"

실제 고객님들의 목소리입니다.
```

**설득 요소:** 증명  
**데이터 반영:** 2단계 FACT(실버 925) + 3단계 리뷰 표현("휘뚜루마뚜루", "안 빼고 자도")
