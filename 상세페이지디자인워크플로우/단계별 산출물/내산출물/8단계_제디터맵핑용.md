## ✅ 필수 이미지 (최소 2-3장)

### 📷 이미지 1: 메인 후킹컷 (가장 중요!)

**촬영 가이드:**
- **분위기**: 세련되고 모던한, 따뜻한 자연광
- **구도**: 모델의 측면 또는 3/4 각도에서 귀걸이가 얼굴선과 함께 보이도록
- **복장**: 깔끔한 베이직 니트 또는 블라우스 (화이트/베이지/그레이 톤)
- **배경**: 화이트 벽 또는 미니멀한 인테리어 공간
- **포인트**: 귀걸이의 웨이브 라인이 명확하게, 얼굴 표정은 자연스럽고 자신감 있게

**예시:**
> 창가에서 자연광을 받으며, 모델이 살짝 고개를 들어 귀걸이가 햇빛에 반짝이는 순간을 포착. 화이트 니트에 젠틀한 미소. 귀걸이가 주인공이지만 모델의 자연스러운 일상 분위기가 느껴지도록.

**목적:** 
- 7단계 [1번 섹션 "빛으로 완성되는 순간"] 시각화
- 첫인상에서 "나도 저렇게 세련되게 착용할 수 있을 것 같아" 느낌 전달
- 브랜드 톤(모던·세련·자신감) 한 장으로 표현

**핸드폰 촬영 팁:**
- 세로 모드, 포트레이트 기능 ON
- 창가에서 오전 10-11시 촬영 (부드러운 자연광)
- 모델과 1.5m 거리 유지

---

### 📷 이미지 2: 착용 클로즈업 (편안함 증명)

**촬영 가이드:**
- **분위기**: 깨끗하고 선명한, 제품 디테일 강조
- **구도**: 귀에 착용된 상태를 클로즈업, 귓볼과 귀걸이의 핏감이 보이도록
- **각도**: 약간 위에서 또는 정면에서 귀 중심으로
- **배경**: 아웃포커싱 (귀걸이만 또렷하게)
- **포인트**: 원터치 구조의 안정감, 무게감이 느껴지지 않는 자연스러운 착용 상태

**예시:**
> 귀에 착용된 귀걸이를 1-2초 정지 상태로 촬영. 귓볼이 편안해 보이고, 귀걸이가 너무 무겁지 않게 자연스럽게 걸려 있는 모습. 웨이브 디테일과 실버 광택이 선명하게.

**목적:**
- 7단계 [3번 "클래스핏 원터치 구조" + 4번 "볼드하지만 가벼운 곡선"] 시각화
- "정말 편하겠다"는 신뢰감 전달
- 3단계 리뷰 부정 키워드("착용 어려움", "무거움") 해소 증명

**핸드폰 촬영 팁:**
- 포트레이트 모드로 배경 흐리기
- 귀걸이에 초점 맞추기 (탭해서 포커스)
- 여러 각도로 10장 이상 촬영 후 베스트 선택

---

### 📷 이미지 3: 라이프스타일 컷 (활용도 증명)

**촬영 가이드:**
- **분위기**: 일상적이고 자연스러운, "나의 평범한 하루"
- **구도**: 모델 전신 또는 상반신, 귀걸이는 살짝 보이는 정도 (주연은 아님)
- **상황**: 카페에서 노트북 작업 / 거울 보며 준비 / 거리 걷기 등
- **복장**: 출근룩 (블라우스+팬츠) 또는 위켄드 룩 (니트+청바지)
- **포인트**: "이 귀걸이가 내 일상에 자연스럽게 녹아 있다" 느낌

**예시:**
> 화이트 셔츠에 베이지 팬츠를 입은 모델이 카페 창가에 앉아 노트북을 보고 있는 장면. 귀걸이가 은은하게 빛나며 전체 룩을 완성. 자연스럽고 세련된 일상.

**목적:**
- 7단계 [7번 "하루의 모든 순간에" + 9번 "당신을 닮은 세련됨"] 시각화
- TPO 범용성 증명 (오피스도 OK, 데일리도 OK)
- "나도 이렇게 입고 싶다" 구매 욕구 자극

**핸드폰 촬영 팁:**
- 일상 스냅 느낌으로 자연스럽게
- 과하게 포즈 잡지 말고 실제 하는 행동을 촬영
- 여러 상황 (출근 준비, 카페, 외출) 3-5컷 준비

## 💡 선택 이미지 (있으면 좋음, 없어도 제디터 AI로 보완 가능)

### 📷 이미지 4: 제품 플랫레이(Flat Lay)
**내용:** 흰 배경에 귀걸이 + 주얼리 케이스 + 작은 소품(꽃, 책 등) 배치  
**목적:** 7단계 [5번 "순도 높은 실버 925"] 증명, 고급스러운 패키징 소개  
**제디터 대체:** AI로 깨끗한 제품 컷 생성 가능

### 📷 이미지 5: 디테일 클로즈업 (925 각인)
**내용:** 실버 925 각인 부분을 매크로 촬영  
**목적:** 7단계 [5번 "순도 높은 실버 925"] 신뢰도 강화  
**제디터 대체:** 제품 정보 텍스트로 대체 가능

### 📷 이미지 6: 비포/애프터 비교
**내용:** 귀걸이 착용 전/후 인상 변화  
**목적:** 7단계 [4번 "볼드하지만 가벼운 곡선"] 효과 시각화  
**제디터 대체:** 단일 착용컷으로 충분

### 📷 이미지 7: 레이어드 스타일링
**내용:** 목걸이, 팔찌와 함께 착용한 전체 코디  
**목적:** 7단계 [7번 "하루의 모든 순간에"] 스타일링 아이디어 제공  
**제디터 대체:** 텍스트 설명으로 가능

## 📝 빠른 촬영 팁

### ✅ 지금 당장 촬영 가능 (15-20분 소요)

**장소:**
- 집 안 창가 (자연광 최고)
- 화이트 벽 앞 (깔끔한 배경)
- 침대 위 화이트 시트 (플랫레이)

**조명:**
- 오전 10-11시 또는 오후 2-3시 간접 자연광
- 실내등 대신 창문 빛 활용
- 귀걸이에 빛 반사되도록 위치 조정

**각도:**
- 메인컷: 측면 3/4 각도
- 클로즈업: 정면 또는 약간 위에서
- 라이프스타일: 자연스러운 일상 각도

**소요 시간:**
- 이미지 1: 5분 (10-20컷 촬영 후 선택)
- 이미지 2: 5분 (클로즈업 여러 각도)
- 이미지 3: 10분 (상황 연출 + 촬영)
- **총 20분이면 3장 확보!**

## 🎯 제디터 입력 시퀀스 (권장 순서)

### STEP 1: 기본 정보 입력
```
1. 상품명 입력 (위 1번 복사)
2. 브랜드명 입력 (위 2번 복사)
3. 핵심 특징 5개 입력 (위 3번 복사)
```

### STEP 2: 카피 입력
```
1. 메인 카피 입력 (위 4번 복사)
2. 서브 카피 A, B 입력 (위 5번 복사)
```

### STEP 3: 이미지 업로드
```
1. 이미지 1 (메인 후킹컷) 업로드 → 상단 배치
2. 이미지 2 (착용 클로즈업) 업로드 → 중단 배치
3. 이미지 3 (라이프스타일) 업로드 → 하단 배치
```

### STEP 4: 스토리 섹션 구성 (제디터 AI 활용)
```
"7단계 스토리 11개 섹션 중 핵심 3-4개만 선택해서 배치"

추천 섹션:
- 섹션 2: "매일 손이 가는 이유" (공감)
- 섹션 3: "클래스핏 원터치 구조" (해결)
- 섹션 5: "순도 높은 실버 925" (증명)
- 섹션 11: "지금, 당신의 빛을 켜세요" (CTA)
```
